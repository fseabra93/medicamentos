Medication List App - Walkthrough
Overview
A web application to create a list of daily medications.

Manual verification: User types and confirms names.
Manual dosage: User types dosages directly.
Image generation: Downloads a clean PNG of the list.

How to Run Locally
Open a terminal in the project folder:
cd /home/flavio/Documentos/agente_medicamentos

Start the development server:
npm run dev
Open the browser at the shown URL (usually http://localhost:3000).
